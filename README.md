## 📋 Project Overview

This is a **production-ready company website** built for Xcellence to sell their cheesy corn chips online. The website includes product showcase, direct WhatsApp ordering, COD information, and social media integration.

### Client Requirements
- Professional brand presence
- Easy online ordering system
- Mobile-responsive design
- Multiple contact options (2 WhatsApp numbers)
- Cash-on-Delivery support for UUM campus

---

| Feature | Description |
